# Portfolio Website

A modern, responsive portfolio website built to showcase projects, skills, and professional experience with a focus on performance, accessibility, and clean UI/UX.

## Features

Responsive, mobile-first design

Project showcase with detailed modals

Smooth animations and modern UI

Cross-browser compatibility

Optimized performance and accessibility

🛠 Tech Stack

Vite

React.js

Tailwind CSS

JavaScript (ES6+)

## Installation & Setup

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Local Development

```bash
git clone this repository url
cd portfolio-website
npm install
npm run dev
```
